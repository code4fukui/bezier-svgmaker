# bezier-svgmaker

ベジェ曲線を使ってSVGグラフィックを作成できるシンプルなWebアプリケーションです。マウスやタッチ操作でコントロールポイントを移動し、曲線を描画できます。生成されたSVGコードをテキストエリアにコピーすることができます。

## 機能
- ベジェ曲線の描画
- コントロールポイントの追加・削除
- コントロールポイントの表示/非表示切り替え
- 線の太さと色の変更
- SVGコードの出力

## 使い方
1. ブラウザで[デモページ](https://taisukefukuno.github.io/bezier-svgmaker/)を開きます。
2. マウスやタッチでコントロールポイントを追加・移動して、ベジェ曲線を描画します。
3. 線の太さと色を変更できます。
4. 「remove point」ボタンでコントロールポイントを削除できます。
5. 「toggle control point」ボタンでコントロールポイントの表示/非表示を切り替えられます。
6. 生成されたSVGコードがテキストエリアに出力されます。
