# bezier-svgmaker

ベジェ曲線を生成してSVGコードを出力するWebツールです。マウスやタッチ操作でコントロールポイントを移動し、ベジェ曲線を描画することができます。生成されたSVGコードはテキストエリアにコピーできます。

## デモ
[bezier-svgmaker](https://fukuno.jig.jp/2314)でデモを確認できます。

## 機能
- ベジェ曲線の描画
- コントロールポイントの追加・削除
- コントロールポイントの表示/非表示切り替え
- 線の太さと色の変更
- SVGコードの出力

## 使い方
1. ブラウザでindex.htmlを開きます。
2. マウスやタッチでコントロールポイントを追加・移動して、ベジェ曲線を描画します。
3. 線の太さと色を変更できます。
4. 「remove point」ボタンでコントロールポイントを削除できます。
5. 「toggle control point」ボタンでコントロールポイントの表示/非表示を切り替えられます。
6. 生成されたSVGコードがテキストエリアに出力されます。
